Course Overview

This course will help you understand how computers work and how to use them. We’ll talk about how to
set up a computer, the difference between hardware and software, and the types of computers you can
use. We’ll also explore operating systems, and basic computing applications

Target Audience

This course is specifically designed for non-technical audiences and/or professionals who may have formal
education in a different discipline, but need to gain more understanding of the basic principles behind
Information & Communications Technologies (ICT)

Course Content

+ Introduction to Computer
  • What Is a Computer?
  • Types of Computers
  • Computer Hardware Components
  • Computer Software
  • Computer Operating System
+ Application Fundamentals
  • Microsoft Word
  • Microsoft Excel
  • Microsoft PowerPoint
